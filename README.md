# Music Player
> #### A Music Player App based on Flutter and Firebase.

## Features
* SignUp/Login (firebase authentication)
* Albums and Songs
* Resume/Pause, Backward/Forward Navigation in Songs Queue and Slider on the Song Page
* MiniPlayer for Songs
